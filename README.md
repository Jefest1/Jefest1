<h2><img src="https://img.shields.io/badge/About%20Me-0f172a?style=for-the-badge&logo=aboutdotme&logoColor=10B981&labelColor=0f172a&color=0f172a" alt="About Me" /></h2>

I am an **AI/ML Engineer** building practical intelligent systems across machine learning, generative AI, backend engineering, and cloud deployment.

My work focuses on turning AI ideas into real, usable products. I build **agentic AI systems, retrieval-augmented applications, tool-enabled conversational workflows, AI automations, and model deployment services** using technologies such as **Python, FastAPI, Docker, Microsoft Azure, LangChain, LangGraph, OpenAI Agents SDK, Google ADK, MCP, and A2A**.

At **4TH-IR**, I work on intelligent systems and automations for business workflows, AI-based data lineage extraction, retrieval and tool-enabled conversational systems, model testing and deployment wrappers, and orchestration logic for specialised AI personas. I also contribute to solution design, project scoping, and production-ready AI prototypes that combine retrieval, prompt orchestration, external APIs, and reliable backend services.

Beyond engineering, I have worked on AI model evaluation, multi-tool agent behavior, dataset curation, RLHF workflows, and OSWorld-style agent evaluation tasks. This gives me a strong practical view of how modern AI systems should reason, use tools, stay grounded, and complete real user workflows.

I am especially interested in building AI solutions for **healthcare, agriculture, finance, business operations, and emerging-market challenges** where intelligent systems can create measurable impact.
